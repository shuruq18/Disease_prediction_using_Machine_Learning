# рџЊЂ Parkinson's Disease Detection from Spiral Drawings

A machine learning project that detects signs of **Parkinson's disease** from hand-drawn spiral images, using **HOG (Histogram of Oriented Gradients)** feature extraction and a **Random Forest** classifier.

> вљ пёЏ **Disclaimer:** This project is for **educational purposes only**. It is not a diagnostic tool and must not be used as a substitute for professional medical advice, testing, or diagnosis.

---

## рџ“Њ Overview

People with Parkinson's disease often show tremors and motor-control difficulties that show up as irregularities in hand-drawn spirals. This project takes an image of a hand-drawn spiral and classifies it as:

- **healthy**, or
- **parkinson**

The pipeline extracts HOG (shape/edge-texture) features from each grayscale image and trains a Random Forest classifier to distinguish the two classes.

## рџ§  Methodology

### 1. Feature Extraction
For every image in each class folder:
- Load the image and convert to grayscale
- Resize to `128x128`
- Extract **HOG features** (`orientations=9`, `pixels_per_cell=(8,8)`, `cells_per_block=(2,2)`, `block_norm="L2-Hys"`)

### 2. Label Encoding
- Folder names (`healthy` / `parkinson`) are encoded to integers with `LabelEncoder`

### 3. Train/Test Split
- 80/20 split, stratified by class, `random_state=42`

### 4. Model Training
- `RandomForestClassifier(n_estimators=200, random_state=42, n_jobs=-1)`

### 5. Evaluation
- Accuracy score
- Full classification report (precision, recall, F1-score) per class
- Confusion matrix, plotted with `ConfusionMatrixDisplay`

### 6. Single-Image Prediction
A helper function, `predict_image(image_path)`, runs the full pipeline (grayscale в†’ resize в†’ HOG в†’ predict в†’ decode label) on a single new spiral image and returns `"healthy"` or `"parkinson"`.

### 7. Saved Artifacts
The trained model and label encoder are persisted with `joblib`:
- `parkinson_rf_model.pkl` вЂ” trained Random Forest model
- `label_encoder.pkl` вЂ” fitted `LabelEncoder` (classes: `healthy`, `parkinson`)

## вљ™пёЏ Installation & Usage

### Requirements
```bash
pip install opencv-python numpy matplotlib scikit-image scikit-learn joblib
```

### 1. Train the model
1. Download/organize the spiral drawing dataset as shown above.
2. Update `DATASET_PATH` in `Parknson_proj.ipynb` to your local dataset path.
3. Run all cells. This extracts HOG features, trains the Random Forest, evaluates it, and saves `parkinson_rf_model.pkl` and `label_encoder.pkl`.

### 2. Predict on a new spiral image
```python
import cv2, joblib
from skimage.feature import hog

rf_model = joblib.load("parkinson_rf_model.pkl")
le = joblib.load("label_encoder.pkl")
IMAGE_SIZE = (128, 128)

def predict_image(image_path):
    img = cv2.imread(image_path)
    img = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
    img = cv2.resize(img, IMAGE_SIZE)
    hog_features = hog(
        img, orientations=9, pixels_per_cell=(8, 8),
        cells_per_block=(2, 2), block_norm="L2-Hys"
    ).reshape(1, -1)
    pred = rf_model.predict(hog_features)
    return le.inverse_transform(pred)[0]

print(predict_image("path/to/your/spiral.png"))
```
